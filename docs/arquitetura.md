# Arquitetura do Sistema

O backend do Rota Livre foi desenvolvido utilizando ASP.NET Core seguindo uma arquitetura em camadas.

Essa organização separa responsabilidades entre os componentes da aplicação, tornando o sistema mais organizado, reutilizável, testável e de fácil manutenção.

Este documento apresenta uma visão geral da arquitetura utilizada no projeto. A documentação detalhada de cada componente será disponibilizada em documentos específicos.
